{THE CODE OF CRAIG}

Code the 1st--

Thou Shalt love the Leader.

Code the 2nd--

Thou Shalt Renounce all earthly ties.

Code the 3rd--

Thou Shalt Renounce thy earthly name.

Code the 4th--

Thou Shalt take the Holy Name of Craig, that the leader hath chosen for us all.

Code the 5th--

Thou Shalt spread the love of the Leader to the world.

Code the 6th--

Thou Shalt keep the secrets of the Church.

Code the 7th--

Thou Shalt Remember the Code, and Adhere to its Word.
